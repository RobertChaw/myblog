# 我的博客 后端

## 项目使用技术

1. Midway框架处理请求
2. Prisma 映射数据库
3. Passport 实现支持github、传统用户登录

配套管理端：[链接][admin]

### Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### Deploy

```bash
$ npm start
```

### npm scripts

- Use `npm run lint` to check code style.
- Use `npm test` to run unit test.
